# Login Page with Animated Loading Effect

This project is a simple login page with an animated loading effect. It features a modern UI with a dark theme, responsive input fields, and an interactive label animation.

# Features

Animated loading effect with blinking elements.

Responsive login form with smooth transitions.

Custom floating labels for input fields.

Dark mode theme with neon blue highlights.

# Technologies Used

HTML

CSS (including Google Fonts: Poppins)
